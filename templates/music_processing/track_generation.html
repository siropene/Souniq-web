Ayuda - Generación de Tracks
Parámetros de Generación:
  • Creatividad (Temperatura): Controla qué tan creativa es la generación
    • 0.1-0.5: Muy conservador, se mantiene cerca del original
    • 0.6-1.0: Equilibrado entre originalidad y creatividad
    • 1.1-2.0: Muy creativo, puede generar variaciones sorprendentes
  • Incluir batería: Añade elementos de percusión a la generación
Configuración automática de Orpheus:
  • Tokens de entrada: 6656 (optimizado)
  • Tokens de generación: 512 (duración equilibrada)
  • Diversidad (Top-p): 0.96 (alta calidad)
  • Limpieza de MIDI: automática
  • Aplicación de sustains: automática
Consejos:
  • Para música similar al original: usa creatividad entre 0.5-0.8
  • Para experimentar y obtener variaciones interesantes: usa 1.0-1.5
  • La batería funciona mejor con géneros rítmicos
  • El proceso puede tardar hasta 15 minutos
